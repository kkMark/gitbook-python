#### 赋值运算符

| 赋值运算符 |                                                              |
| ---------- | ------------------------------------------------------------ |
| =          | 赋值                                                         |
| +=         | 加法赋值                                                     |
| -=         | 减法赋值                                                     |
| *=         | 乘法赋值                                                     |
| /=         | 除法赋值                                                     |
| %=         | 取余赋值                                                     |
| **=        | 幂赋值                                                       |
| //=        | 取整除赋值                                                   |
| :=         | 海象运算符，可在表达式内部为变量赋值。**Python3.8 版本新增运算符** |

```python
# 不使用海象运算符
list = [1, 2, 3]
count = len(list)
if count > 3:
   print(f"Error, {count} is too many items")

# 海象运算符
list = [1, 2, 3]
if (count := len(list)) > 3:
   print(f"Error, {count} is too many items")
```

#### 位运算符

| 位运算符 |                                               |
| -------- | --------------------------------------------- |
| &        | 按位与运算符，二进制同为1则1，否则为0         |
| \|       | 按位或运算符，二进制有一个为1，则为1，否则为0 |
| ^        | 按位异或运算符，二进制不同则为1               |
| ~        | 按位取反运算符，二进制位取反                  |
| <<       | 左移动运算符                                  |
| >>       | 右移动运算符                                  |

```python
# 0011 1100
a = 60	
# 0000 1101
b = 13

"""
a + b = c
&: 与运算符
0011 1100 & 0000 1101 = 0000 1100
"""

"""
a | b = c
|: 或运算符
0011 1100 | 0000 1101 = 0011 1101
"""

"""
a ^ b = c
^: 异或运算符
0011 1100 ^ 0000 1101 = 0011 0001
"""

"""
~a = c
~: 取反运算符
~0011 1100 = 1100 0011
"""

"""
左移动运算符
a << 2 = c
0011 1100 << 2 = 1111 0000
"""

"""
右移动运算符
a >> 2 = c
0011 1100 >> 2 = 0000 1111
"""
```
